LocalStorage (ES5)

Gracias a la propiedad LocalStorage de JavaScript podemos acceder al almacenamiento interno del navegador

Los datos guardados son persistentes al cierre de la página o del navegador

Aplicación serverless, todo se ejecuta en el cliente sin necesidad de servidor

Posibilidad de descargar y cargar la base de datos en un archivo JSON

Se ha desarrollado en JavaScript versión ECMAScript5 (2009) por termas de retrocompatibilidad:

  • Chrome 23
  • IE10/Edge
  • Firefox 21
  • Safari 6
  • Opera 15
